Sales Aggregation Tool
利用マニュアル

販売管理システムからのXLS/CSVデータを読み込み、カテゴリ修正・集計・CSV出力を行うツールです。

1ツールについて

sales-aggregation-tool.html の1ファイルのみで動作します。サーバーやインターネット接続は不要で、HTMLファイルをブラウザで開くだけで使用できます。アップロードしたデータは外部に一切送信されません。

ℹ
動作環境 Google Chrome / Microsoft Edge / Safari などモダンブラウザで動作します。Internet Explorerは非対応です。

2機能一覧

📂
ファイル読み込み
XLS / XLSX / CSV に対応。ドラッグ＆ドロップまたはクリックで読み込み。
🏷
カテゴリ修正
商品ごとにカテゴリをプルダウンで変更可能。変更行は緑色でハイライト表示。
⚙️
カテゴリ管理
カテゴリの追加・編集・削除・並べ替えをアプリ内で行える。
📊
集計
カテゴリ×税区分で売上金額を集計。消費税額を参考表示。
⬇️
CSV出力
商品明細・集計結果をそれぞれCSVでダウンロード可能。
🏠
自家用の識別
自家用取引を集計結果で括弧書き表示（うち自家用 ¥XX）。

3基本的な使い方

1
HTMLファイルをブラウザで開く

sales-aggregation-tool.html をダブルクリックしてブラウザで開きます。

2
ファイルを読み込む

アップロードエリアをクリックしてファイルを選択するか、ファイルをドラッグ＆ドロップします。XLS・XLSX・CSV に対応しています。

3
カテゴリを確認・修正する

読み込まれた商品一覧が表示されます。カテゴリ欄がプルダウンの行は変更できます。固定カテゴリ（清酒・リキュール・運賃）はバッジ表示で変更不可です。変更した行は緑色でハイライトされます。

4
「集計する」ボタンを押す

カテゴリ×税区分で集計された結果が表示されます。清酒・リキュールには自家用分の内訳も括弧書きで表示されます。

5
CSVをダウンロードする（任意）

商品一覧・集計結果それぞれの右上にある「CSV出力」ボタンからダウンロードできます。

4カテゴリ管理

ファイル読み込みエリアの下にある「カテゴリを管理する」ボタンを押すと管理パネルが開きます。

カテゴリの追加

パネル下部の入力欄に新しいカテゴリ名を入力し「追加」ボタンを押します。Enterキーでも追加できます。追加されたカテゴリは「その他」の直前に挿入されます。

カテゴリ名の編集

リスト内のカテゴリ名を直接クリックして編集できます。フォーカスを外すと確定されます。固定カテゴリ（清酒・リキュール・運賃）は編集できません。

カテゴリの並べ替え

各行の左端にある「⠿」アイコンをドラッグして順番を変えられます。並べ替えは集計結果テーブルの表示順にも反映されます。

カテゴリの削除

各行の右端の「✕」ボタンで削除できます。固定カテゴリは削除できません。

⚠
適用前の注意 変更後は必ず「適用する」ボタンを押してください。「×」やキャンセルで閉じると変更は破棄されます。削除したカテゴリが商品に割り当てられている場合、自動的に「その他」に変更されます。

5CSV出力

ボタンの場所 ファイル名 出力内容
商品一覧テーブル右上 sales_detail_YYYYMMDD.csv 商品名称・酒別名称・取引名称・カテゴリ（修正後）・税区分・当年金額
集計結果テーブル右上 sales_summary_YYYYMMDD.csv カテゴリ・税区分・売上金額合計・うち自家用・消費税（参考）＋合計行
✓
Excelで文字化けしません 出力CSVはUTF-8 BOM付きで保存されるため、Excelで開いても日本語が正しく表示されます。

6カテゴリルール仕様

酒別名称 取引名称 カテゴリ 種別
清酒 売上 / 自家用 清酒 固定
リキュール 売上 / 自家用 リキュール 固定
運賃 — 運賃 固定
セット品・その他副資材等・値引・その他 — 初期値は酒別名称（変更可） 変更可

自家用取引の清酒・リキュールは固定カテゴリのまま集計されますが、集計結果では「うち自家用」として金額の内訳が括弧書きで表示されます。

7税区分・消費税の仕様

消費税率名称の値 表示ラベル 消費税計算
0.1 10%（標準） 売上合計 × 0.1（四捨五入）
0.08 8%（軽減） 売上合計 × 0.08（四捨五入）
空欄 / null 対象外 —（輸出等）

消費税額は参考値として集計結果に表示されます。端数処理は四捨五入です。

8入力ファイル形式

販売管理システムから出力されたXLS・XLSX・CSVファイルを想定しています。以下の列が含まれている必要があります（列順不問）。

列名 内容 必須
取引名称「売上」「自家用」など必須
酒別名称「清酒」「リキュール」など必須
商品名称商品の名称必須
消費税率名称0.1 / 0.08 / 空欄必須
当年金額売上金額（数値）必須
ℹ
上記以外の列（取引コード・担当者名称など）は無視されます。ファイル先頭に集計期間などのタイトル行がある場合も自動的に読み飛ばします。
