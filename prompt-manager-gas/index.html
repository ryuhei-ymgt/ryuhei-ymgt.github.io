スプレッドシートの準備
データベースとなるスプシを作成します
1
Google スプレッドシートを新規作成する
▼

Google ドライブを開き、「新規」→「Google スプレッドシート」で新しいスプシを作成してください。

シート構成は以下の2シートを作成します（GASが自動作成するので手動作成は不要です）。

シート名用途
templatesテンプレートデータ（id / category / title / prompt / visible）
config管理者パスワード（admin_password）
📋 シートはGASが初回アクセス時に自動作成します。手動で作る必要はありません。
2
スプレッドシートIDを確認する
▼

スプレッドシートのURLから /d/ と /edit の間の文字列がIDです。

https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit ↑ この部分がスプレッドシートID

このIDは次のGAS設定で使います。メモしておいてください。

2
Google Apps Script の設定
GASをWebアプリとしてデプロイします
1
Apps Script を開く
▼

スプレッドシートのメニューから「拡張機能」→「Apps Script」をクリックしてください。

📌 メニューバー：拡張機能 → Apps Script

新しいタブでApps Scriptエディタが開きます。

2
コードを貼り付ける
▼

エディタに最初から書かれているコードをすべて削除して、以下のコードをそのまま貼り付けてください。

// ============================================================ // プロンプト管理アプリ - GAS側コード // ============================================================ // 【設定】スプレッドシートIDをここに入力 const SPREADSHEET_ID = "ここにスプレッドシートIDを入力"; const SHEET_NAME_TEMPLATES = "templates"; const SHEET_NAME_CONFIG = "config"; // GETリクエスト（読み取り） function doGet(e) { const action = e.parameter.action; try { if (action === "getTemplates") return jsonResponse(getTemplates()); if (action === "getPassword") return jsonResponse(getPassword()); return jsonResponse({error: "unknown action"}); } catch(err) { return jsonResponse({error: err.message}); } } // POSTリクエスト（書き込み） function doPost(e) { try { const body = JSON.parse(e.postData.contents); const action = body.action; if (action === "saveTemplates") { saveTemplates(body.templates); return jsonResponse({ok: true}); } if (action === "savePassword") { savePassword(body.password); return jsonResponse({ok: true}); } return jsonResponse({error: "unknown action"}); } catch(err) { return jsonResponse({error: err.message}); } } // テンプレート取得 function getTemplates() { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); const sheet = ss.getSheetByName(SHEET_NAME_TEMPLATES); if (!sheet) return {templates: []}; const data = sheet.getDataRange().getValues(); if (data.length < 2) return {templates: []}; const templates = data.slice(1) .filter(row => row[0] && row[2] && row[3]) .map(row => ({ id: String(row[0]).trim(), category: String(row[1] || "未分類").trim(), title: String(row[2]).trim(), prompt: String(row[3]).trim(), visible: String(row[4]).trim() === "false" ? false : true })); return {templates}; } // テンプレート保存（上書き） function saveTemplates(templates) { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); let sheet = ss.getSheetByName(SHEET_NAME_TEMPLATES); if (!sheet) sheet = ss.insertSheet(SHEET_NAME_TEMPLATES); sheet.clearContents(); const rows = [["id", "category", "title", "prompt", "visible"]]; templates.forEach(t => rows.push([t.id, t.category, t.title, t.prompt, t.visible === false ? "false" : "true"])); sheet.getRange(1, 1, rows.length, 5).setValues(rows); } // パスワード取得 function getPassword() { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); const sheet = ss.getSheetByName(SHEET_NAME_CONFIG); if (!sheet) return {password: ""}; const data = sheet.getDataRange().getValues(); const row = data.find(r => String(r[0]).trim() === "admin_password"); return {password: row ? String(row[1]).trim() : ""}; } // パスワード保存 function savePassword(password) { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); let sheet = ss.getSheetByName(SHEET_NAME_CONFIG); if (!sheet) { sheet = ss.insertSheet(SHEET_NAME_CONFIG); sheet.appendRow(["key", "value"]); } const data = sheet.getDataRange().getValues(); const rowIdx = data.findIndex(r => String(r[0]).trim() === "admin_password"); if (rowIdx >= 0) { sheet.getRange(rowIdx + 1, 2).setValue(password); } else { sheet.appendRow(["admin_password", password]); } } // レスポンスヘルパー（CORS対応） function jsonResponse(data) { return ContentService .createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); }
⚠️ 既存の function myFunction(){} は必ず削除してから貼り付けてください。

貼り付けたら、1行目の SPREADSHEET_ID を自分のスプシIDに書き換えます。

// 変更前 const SPREADSHEET_ID = "ここにスプレッドシートIDを入力"; // 変更後（例） const SPREADSHEET_ID = "1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms";

変更したら 💾（保存）ボタンまたは Ctrl+S で保存してください。

3
Webアプリとしてデプロイする
▼

右上の「デプロイ」→「新しいデプロイ」をクリックしてください。

📌 デプロイ（右上青ボタン）→ 新しいデプロイ

設定項目を以下のように設定してください。

項目設定値
種類の選択ウェブアプリ
説明任意（例：プロンプト管理API）
次のユーザーとして実行自分
アクセスできるユーザー全員
⚠️ 「アクセスできるユーザー」は「全員」に設定してください。「自分のみ」だとHTMLからアクセスできません。

「デプロイ」ボタンをクリックするとGoogleアカウントの認証が求められます。「権限を確認」→「許可」を選択してください。

✓ デプロイが完了すると WebアプリのURL（https://script.google.com/macros/s/.../exec）が表示されます。必ずコピーしておいてください。
💡 取得したWebアプリのURLは、スプレッドシートの Sheet1のA1セル などにメモしておくと紛失防止になります。再デプロイでURLが変わった場合もすぐ確認できて便利です。
4
コード変更後の再デプロイについて
▼

GASのコードを変更した場合、「新しいデプロイ」ではなく「デプロイを管理」→ 既存のデプロイを編集（鉛筆アイコン）→「バージョン：新しいバージョン」で更新してください。

⚠️ 「新しいデプロイ」を毎回実行するとURLが変わってしまいます。既存デプロイの更新を使うとURLは変わりません。
3
アプリの初期設定
GAS URLを登録して動作確認します
1
アプリを開いてログインする
▼

管理者からお伝えしたアプリのURLをブラウザで開いてください。初回はGAS URLが未設定のためログイン画面が表示されます。

初期パスワードは admin123 です。ログイン後すぐに変更することを推奨します。

⚠️ ローカルの index.html をダブルクリックで開いても動作しません。必ず管理者から共有されたURL（https://...）からアクセスしてください。
⚠️ ブラウザのキャッシュ・履歴を削除した場合、GAS URLがリセットされるため初期パスワード（admin123）でログインできる状態になります。ただしGAS URLが未設定のためスプシに接続されず、実際のテンプレートデータは表示されません。設定タブでGAS URLを再入力することで通常運用に戻ります。
2
設定タブでGAS URLを登録する
▼

管理者でログイン後、右上の「設定」タブを開いてください。

📌 管理者画面 → 設定タブ → 「GAS WebアプリURL」欄

PHASE 2 でコピーしたGAS URLを貼り付けて「URLを保存して再接続」をクリックしてください。

✓ 「接続成功！テンプレート〇件を読み込みました」と表示されれば設定完了です。
3
パスワードを変更する
▼

設定タブの「管理者パスワードの変更」セクションでパスワードを変更してください。変更するとスプレッドシートの config シートに保存され、次回アクセス時から新しいパスワードが有効になります。

⚠️ 初期パスワード（admin123）のままにしておくとセキュリティ上危険です。必ず変更してください。
4
トラブルシューティング
▼
症状原因・対処
接続失敗と表示されるGAS URLが正しいか確認。デプロイ時の「アクセスできるユーザー」が「全員」になっているか確認
データが反映されないGASコードを変更した場合は「デプロイを管理」→既存デプロイを更新（新しいバージョン）する
ログインできない2回目以降はスプシのconfigシートのパスワードが使われる。スプシを直接確認して確認
前回のデータが出ないGAS URLはlocalStorageに保存される。別ブラウザ・別PCでは再設定が必要
URLを教えてもらっていない管理者にアプリのURLを確認してください
キャッシュクリア後に初期PW（admin123）でログインできてしまうGAS URLがリセットされているため。設定タブでGAS URLを再入力すると通常運用に戻ります。GAS URL未設定中は実データは表示されません
